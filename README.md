# My Personal Portfolio

A clean, responsive personal portfolio website built with HTML5, CSS3, and JavaScript. This project showcases my skills, education, and projects as a Web Developer.

## Features

- **Responsive Design**: Adapts seamlessly to different screen sizes (mobile, tablet, desktop).
- **Navigation**:
    - Sticky navigation bar with smooth scrolling to sections.
    - Mobile-responsive hamburger menu.
- **Hero Section**: Introduces the developer with a clear call-to-action.
- **About Me**: Brief introduction and professional summary.
- **Education**: Timeline-style display of educational background.
- **Projects**: Grid layout showcasing recent work with links to live demos.
- **Contact Form**: A functional-looking contact form (demo).
- **Social Links**: Links to GitHub and Instagram profiles.

## Technologies Used

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling with Flexbox and Grid layouts, CSS variables, and media queries.
    - Font: 'Inter' from Google Fonts.
    - Icons: Font Awesome.
- **JavaScript**: Interactive elements including mobile menu toggle and smooth scrolling handling.

## Project Structure

```
/

```


    - Navigate to the project folder.
    - Open `index.html` in your web browser.
